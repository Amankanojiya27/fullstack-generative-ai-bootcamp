Create Virtual Environment (venv)

Commands:
Step 1: Create the virtual environment
    python -m venv .venv

Step 2: Activate the Virtual Environment using the command:
    - For PowerShell: .venv\Scripts\activate
    - For Command Prompt: .venv\Scripts\activate.bat

Step 3: Create a requirements.txt file in the project root directory

Step 4: Install packages from requirements file:
    pip install -r requirements.txt
    or
    pip install -r .\requirements.txt
    
    This will install all packages listed in the requirements.txt file into our virtual environment.
